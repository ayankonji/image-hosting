# 图片存储目录

此目录用于存储上传的图片文件。

## 使用方法

1. 将图片文件放入此目录
2. 使用上传脚本上传到GitHub
3. 通过GitHub Pages访问图片

## 图片命名规范

- 使用英文和数字
- 避免特殊字符
- 建议使用有意义的名称

## 访问URL格式

上传后，图片可通过以下URL访问：

```
https://<username>.github.io/image-hosting/images/<filename>
```

## 示例

- `photo.jpg` → `https://username.github.io/image-hosting/images/photo.jpg`
- `image_001.png` → `https://username.github.io/image-hosting/images/image_001.png`

## 注意事项

- 图片文件大小建议不超过10MB
- 支持常见图片格式：JPG、PNG、GIF、WebP
- 文件名请使用小写字母和数字
